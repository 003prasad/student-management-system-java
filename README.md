# Student Management System – using : Java, mysql

A beginner-level Java full stack web application developed to understand
CRUD operations, JDBC connectivity, and basic web development using Servlets and JSP.

---

## Technologies Used
- Java (Core Java)
- JDBC
- Servlets & JSP
- MySQL
- HTML, CSS
- Apache Tomcat

---

## Features
- Admin Login (Session-based authentication)
- Add Student
- View Students
- Update Student
- Delete Student
- Success/Error messages
- Clean and simple UI 

---

## Database Design
**Database Name:** sms  

**Tables:**
- `students`
- `admin`
